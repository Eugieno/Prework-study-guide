Git

  • git helps developers to keep track of changes in their code
  • git status: checks the current branch we are working on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Typically, git workflow begins from creating a new branch up until merging the new branch feature to the main branch
  • All git commands are preceeded by the word: git
